# Skrypty

Repozytorium z różnymi przydatnymi skryptami w bashu.

* [lun_format.sh](https://github.com/mateusznitka/scripts/blob/main/lun_format.sh) - Formatowanie wielu LUNów do XFS i odczyt UUID filesystemu (do /fstab i dokumentacji)
* [lun_scan.sh](https://github.com/mateusznitka/scripts/blob/main/lun_scan.sh) - Sprawdzanie czy na LUNach jest filesystem (read only)
* [lun_remove.sh](https://github.com/mateusznitka/scripts/blob/main/lun_remove.sh) - Usuwanie nieużywanych LUN z systemu (przed odprezentowaniem z macierzy)
